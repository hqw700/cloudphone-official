Core Features

重新定义网页云端操作体验

抛弃传统 VNC 的卡顿与模糊，在任意设备的浏览器中，体验几乎零延迟的 Android 虚拟机系统。

⚡

硬件级 PTS 透传 (HW-PTS)

直接捕获 Android MediaCodec 编码器的原始 PTS 时间戳计算 RTP Duration，解决时间戳累积漂移，使 Jitter Buffer 降至毫秒级。

🔒

三通道 UDS 隔离

视频流 (@video)、控制流 (@control) 和多点触控 (@touch) 采用三路独立 Unix Domain Socket 连接，彻底杜绝高频触控抢占视频带宽。

🎵

高音质与视频直通

独家支持 Opus 音频直通编码（StreamID: audio_0）并与视频轨道分离渲染。同时支持 H.264 Annex B 帧直接封包，消除重转码延迟。

🌐

公网 P2P 穿透增强

自动探测宿主机网卡并申请 UPnP UDP 端口映射，配合 IPv6 单播直连技术与 STUN/TURN 中继，跨越 CGNAT 屏障，实现公网免配置秒级直连。

💻

原生 WebADB 交互

不依赖本地 ADB。通过 WebRTC 独立的 adb-channel 数据通道直连虚机 5555 端口，配合前端 xterm.js，带来响应时间小于 10ms 的控制台体验。

📈

拥塞控制与参数热更新

集成了 Pion GCC 带宽评估算法，支持 MediaCodec 底层参数热更新，运行时动态改变码率与分辨率，弱网下依然保持画面丝滑连贯。

Get Started

三步开启您的云手机之旅

CloudPhone 原生支持在 MacOS、Linux 和 Windows 上交叉编译并一键运行。跟随以下极简流程，立刻启动您的服务。

阅读完整部署文档
bash - Terminal
# 1. 启动信令服务器
./start_server.sh v2
# 2. 网页直连物理手机进行 Agent 一键部署
# 打开浏览器访问 http://localhost:8443 点击「部署新设备」即可
# 3. 宿主机命令行脚本手动推送 (真机/Docker 适用)
cd agentd && ./run.sh -id phone-01 -signaling ws://127.0.0.1:8443
