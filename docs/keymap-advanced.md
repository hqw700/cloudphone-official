# 自定义按键映射规范

CloudPhone 内置了强大的可视化改键引擎（Keymap Engine），支持在投屏窗口上直接监听键盘事件，并将其转换为高精度的多点触控与滑动轨迹，极大地提升了在网页端操作移动应用与动作游戏的手感。

---

## ⌨️ 映射节点 JSON 配置属性

所有的自定义键位映射规则均可通过 Profile 导出/导入。每个节点遵循以下 JSON 声明规范：

### 1. 单击按键 (Tap)
当按下键盘指定物理键时，在屏幕特定比例坐标注入点击动作：

```json
{
  "type": "tap",
  "key": "K",
  "x": 0.825,
  "y": 0.640,
  "pointerId": 8
}
```

* **`key`**：物理键盘按键名称（大小写敏感，如 `F`、`Space`、`Shift`、`Control`）。
* **`x`, `y`**：横纵轴百分比比例（`0.0` 到 `1.0`），以此完美防黑边拉伸变形。
* **`pointerId`**：指定该指令的物理指针 ID（一般取 `1` 到 `9` 的独立值，不与真实鼠标 `-1` 冲突，从而实现纯键盘多指操作）。

---

### 2. 虚拟摇杆 (Joystick)
将经典的键盘 `W`, `A`, `S`, `D` 物理按键，实时运算转化为以中心为圆心、带有向外滑动拖拽的 360 度跑动模拟向量。

```json
{
  "type": "joystick",
  "keys": {
    "up": "W",
    "left": "A",
    "down": "S",
    "right": "D"
  },
  "centerX": 0.185,
  "centerY": 0.720,
  "radius": 0.085,
  "pointerId": 0
}
```

* **`centerX`, `centerY`**：虚拟轮盘中心在屏幕中的比例坐标。
* **`radius`**：跑动滑动的范围半径比例。
* **`pointerId`**：摇杆占用的专用触控点。摇杆按下不放时，该 ID 将持续在屏幕上模拟 ACTION_MOVE。

---

### 3. 滑动按键 (Swipe)
将单次物理按键触发为从起点到终点、带自定持续时间的单向快速滑动（非常适用于翻页、横版躲避等）：

```json
{
  "type": "swipe",
  "key": "E",
  "startX": 0.5,
  "startY": 0.5,
  "endX": 0.5,
  "endY": 0.2,
  "durationMs": 150,
  "pointerId": 9
}
```

* **`durationMs`**：滑动的持续时间，单位毫秒。时间越短，在 Android 中判定为快速划动（Fling）的概率越高。
